Junior Web Developer


Безкоштовно почати курс

З чого складається курс?

Модуль 1: Основи веб розробки

Вивчаємо як працює веб - фундаментальні основи мережі, особливості роботи браузера, що таке клієнт / сервер, хто за що відповідає і як з тим працювати, HTTP запити. Ці знання забезпечать розуміння середовища з яким і в якому працює Web Developer.

Модуль 2: Основи HTML

Знайомимося з HTML - мовою розмітки гіпертекстових документів що являється стандартом в Інтернеті, вивчаємо синтаксис, структуру розмітки та основні компоненти (теги). Розкриємо поняття документ. Проходимо основи HTML5 - семантичний підхід, нові теги та API.

Модуль 3: Основи CSS

Вивчаємо мову CSS (каскадні таблиці стилів) за допомогою чого ми зможемо будь-яким елементам на сторінці давати гарні стилі. Якщо HTML це структура документа, то CSS це гарне відображення елементів. Кольори, анімації, позиціонування - це все CSS.

Модуль 4: Основи JavaScript

Вивчаємо основи веб-програмування на мові JavaScript щоб зробити сторінку динамічною. Якщо HTML/СSS відповідають за верстку сторінок, то JavaScript це програмна логіка що відповідає за інтерактивність. Що робити коли нажали на кнопу, коли хочемо відіслати дані з форми, і перевірити чи правильно заповнили, як поміняти дані в залежності від дій користувача, ми це все вивчемо на даному етапі, ознайомимось з самою мовою та її фреймворками.

Модуль 5: Agile/SCRUM, GIT

Вивчаємо робочий процес. Крім верстки та програмування ви дізнаєтеся як працювати з кодом, ми навчимо вас працювати з GIT - системою контролю версій де будемо зберігати проекти та повертати до будь-якого стану при потребі, відслідковувати прогрес та виконані завдання, ознайомимось з робочим процесом Agile/SCRUM - розберемо як працювати в команді, планувати, розбивати та виконувати завдання на проектах. Це основні процеси по яких програмісти працюють кожен день, отже ми вас цьому навчимо.

Модуль 6: Підготовка/пошук вакансії

До цього моменту ви вивчите технічну сторону, але що робити далі? Як позиціонувати себе на ринку, де шукати роботу, які компанії бувають, як і куди подаватися, як створити круте резюме, що писати в досвід якщо його ще нема, як показати рекрутеру що ти кращий за конкурентів, заповнюємо правильно основні профілі та набиваємо базу контактів рекрутерів. Публікуємо та заповнюємо портфоліо з першими проектами. Складемо вам план кар'єри, дамо поради по працевлаштуванню та пошуку роботодавців вдома та стажувань за кордоном, це все реально, ми покажемо що і як робити покроково!

Модуль 7: Тестова співбесіда

Ми проведемо вам тестову співбесіду, саме таку яку будуть проводити потенційні роботодавці але в спокійній обстановці, визначимо всі ваші сильні та слабкі сторони, складемо детальний план додаткового навчання та подальших дій, дамо всі необхідні документи, матеріали та посилання а також загальні рекомендації та відповіді на запитання.
